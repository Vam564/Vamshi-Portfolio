  • Cricket Fever
    Live Streaming Web Application

    Cricket Fever is a full-stack live cricket streaming web application I developed to combine my passion for sports with my technical skillset. The platform features a responsive, subscription-based model that enables users to stream matches in real time with smooth performance across devices. I built the front end using React (with Vite), TypeScript, Tailwind CSS, and Redux, focusing on clean UI and mobile-first responsiveness. On the backend, I used Node.js and Supabase with PostgreSQL to manage user authentication, role-based access control, and secure data storage. The application is deployed on Vercel with an automated CI/CD pipeline for seamless updates and fast load times. Real-time data interactions were a core feature, and I ensured that content delivery remained consistent under load.
